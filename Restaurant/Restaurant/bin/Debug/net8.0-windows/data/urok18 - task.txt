делаем приложение для ресторана - учет заказов и продукции, создание сметы.
в приложение нужно добавлять список продуктов, который сохраняется в .txt файлике. в отдельном файле будет список блюд с указанием доступного количества, которое расчитывается исходя из рецепта и доступным продуктов. для добавления блюда нужно добавить список продуктов с указанием количества, которое используется для его приготовления. данный список можно реализовать с помощью динамически генерируемых комбобоксов, в которых можно выбрать продукт, а справа будет маскет_текстБокс, в который вносится количество. доступные для заказа блюда автоматически высвечиваются списком в главном меню, как и список доступных продуктов. так же должен быть список сделаных за текующую смену заказов: сколько посетителей, блюд, стоимость и тд. показатель общей суммы за смену.
